# FDM_Mini_Project_Frontend
MovieDB, a Machine Learning and Data Mining-aided Intelligent Movie Database Rich Webapp.

### Prerequisits
- jdk 8
- jre 1.8
- maven 3.x
- JAX-RS/Jersey 1.19.x
- MySQL Connector
